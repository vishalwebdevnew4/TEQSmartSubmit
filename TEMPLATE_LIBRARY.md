## Auto-Selection Logic

The system automatically selects templates based on business categories:

- **Restaurant/Food** → `restaurant` template
- **Medical/Health** → `medical` template
- **Real Estate** → `real-estate` template
- **Default** → `premium-modern` template

## Dynamic Data Injection

All templates automatically inject:
- Business name
- Address
- Phone number
- Website URL
- Description
- Services/Categories
- Contact information

## Template Sources

1. **GitHub Templates**: Downloads from popular Next.js repositories
2. **Built-in Templates**: Pre-generated templates (instant use, no download needed)

## Usage

When generating a website, the system:
1. Detects business category
2. Selects appropriate template
3. Injects business data
4. Generates complete Next.js project
